# learning-to-recall-experiments

Code for **Learning to Recall with Transformers Beyond Orthogonal Embeddings** *Nuri Mert Vural, Alberto Bietti, Mahdi Soltanolkotabi* (ICLR 2026).

This repository contains the lightweight Jupyter notebooks and saved results used to generate the figures in the paper.

## Repository Contents

* **`one_layer_attention_mlp.ipynb`**: Code for three-step gradient descent experiments (Figures 1a, 2, 3, 4)
* **`one_layer_attention_adam.ipynb`**: Code for multi-parent experiments (Figures 5, 6, 7)

## Saved Results

* **`results/three_step_GD/`**: Saved results for three-step gradient descent experiments (Figures 1a, 2, 3, 4)
* **`results/multi_step_adam/`**: Saved results for multi-step Adam experiments (Figures 5, 6, 7)
